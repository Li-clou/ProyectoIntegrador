Características

Búsqueda Inteligente

Encuentra códigos postales por colonia, ciudad, municipio o estado por coincidencias.

API REST Rápida

Respuestas en milisegundos con datos actualizados a 2025. Con una estructura JSON sencilla.

100 % Open Source

Código abierto. Contribuye al proyecto y ayuda a mejorar los datos.

Actualizaciones en un instante

Con los Scripts integrados, los datos se actualizan en un instante.

Por qué usar México API?

México API es una herramienta poderosa para desarrolladores y organizaciones que necesitan acceder a datos de códigos postales, cuadros delimitadores de estados (Bbox), estados, municipios, ciudades y colonias de México. Con una API REST rápida y fácil de usar, puedes integrar información geográfica precisa en tus aplicaciones y servicios.

Búsca y pruéba.

Ingresa un Código Postal para obtener información geográfica.
